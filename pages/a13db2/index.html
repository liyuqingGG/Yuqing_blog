unittest基础原创

# unittest四大组件

  1. TestCase 测试用例
    • 定义一个测试用例类，必须继承unittest模块中的TestCase类
    • 测试用例类中，一个test开头的就是一条测试用例
    • 用例执行逻辑编写
  2. TestSuite 测试套件
  3. TestRunner 测试运行 
import unittest
from unittestreport import TestRunner
# 创建测试套件收集测试用例类
suite = unittest.defaultTestLoader.discover('TestCase文件夹路径')
runner = TestRunner(suite)
runner.run()
